# 📚 提默小助手

一款专为小学生设计的字词默写练习网页应用，模拟老师课堂提默场景，支持语音朗读、语音输入和自定义间隔时间。

![提默小助手](https://placehold.co/600x300/4CAF50/FFFFFF?text=提默小助手)

## ✨ 功能特点

- 🎤 **语音输入** - 通过麦克风语音输入要默写的词语
- 🔊 **语音朗读** - 自动朗读当前要默写的词语（支持中英文）
- 📝 **词语管理** - 添加、编辑、删除单个词语，批量粘贴
- ⏱️ **自定义间隔** - 可调节提默间隔时间（3-30秒）
- ✍️ **默写模式** - 全屏专注模式，显示进度和当前词语
- ✅ **对照检查** - 完成后可标记对错，统计正确率
- 💾 **本地存储** - 自动保存设置和常用词库
- 📱 **响应式设计** - 支持电脑、平板使用

## 🚀 快速开始

### 方式一：直接打开（推荐）

1. 下载本项目到本地
2. 双击 `index.html` 文件，用浏览器打开即可使用

### 方式二：本地服务器

```bash
# 进入项目目录
cd write-words

# 使用 Python 启动本地服务器
python -m http.server 8080

# 或使用 Node.js
npx serve .

# 然后访问 http://localhost:8080
```

## 📖 使用说明

### 1. 添加词语

**文字输入：**
- 在输入框中输入词语，按回车或点击"添加"按钮
- 支持批量粘贴多个词语（用空格、逗号分隔）

**语音输入：**
- 点击麦克风按钮 🎤
- 说出要添加的词语
- 识别成功后自动添加到列表

**批量粘贴：**
- 点击"+ 批量粘贴"
- 在文本框中粘贴多个词语（可用空格、逗号、换行分隔）
- 点击"批量添加"

### 2. 管理词语

- **编辑**：点击词语旁的 ✏️ 按钮，修改后保存
- **删除**：点击词语旁的 🗑️ 按钮删除单个词语
- **清空**：点击"清空全部"按钮删除所有词语

### 3. 设置选项

- **提默间隔时间**：拖动滑块设置（3-30秒）
- **朗读语音**：选择喜欢的语音（支持中英文）
- **显示词语**：勾选则在提默时显示词语（适合跟读练习）

### 4. 开始提默

1. 点击"开始提默"按钮
2. 等待 3 秒倒计时
3. 系统会自动朗读第一个词语
4. 按照设定的间隔时间自动切换下一个
5. 可随时暂停、继续、跳过或重复

### 5. 检查答案

1. 提默完成后进入检查页面
2. 对照自己默写的结果
3. 点击"✓ 对"或"✕ 错"标记每个词语
4. 查看正确率和统计信息

### 6. 快速词库

点击下方的快捷按钮快速加载预设词库：
- 数字 1-10
- 星期
- 颜色
- 动物

## 🔧 技术栈

- **前端**：纯 HTML5 + CSS3 + JavaScript（无框架依赖）
- **样式**：Tailwind CSS + 自定义样式
- **语音**：Web Speech API（语音合成 + 语音识别）
- **存储**：localStorage 本地存储

## 🐛 常见问题

**Q: 语音朗读没有声音？**
A: 请检查：
1. 设备音量是否开启
2. 浏览器是否有播放权限
3. 是否选择了正确的语音

**Q: 语音输入无法使用？**
A: 语音输入需要：
1. Chrome 或 Edge 浏览器
2. 麦克风权限（点击允许）
3. 网络连接（部分语音识别需要联网）

**Q: 如何保存自定义词库？**
A: 词库会自动保存到浏览器本地存储，下次打开时仍然保留。

**Q: 可以在手机上使用吗？**
A: 可以！应用采用响应式设计，支持手机浏览器访问。但语音功能在 iOS Safari 上可能受限。
